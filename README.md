## Installation

### Prerequisites

- .NET 9 SDK

### Setup

  ```bash
  dotnet restore
dotnet run --project CsvBuddy
``` 

### Testing
  ```bash
  dotnet test
``` 

## Features

- Parsing & Editing: Load, edit, and save CSV files.
- Cross-Platform GUI: Built with Avalonia UI, runs on Windows, macOS, and Linux.
